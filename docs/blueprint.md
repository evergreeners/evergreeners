# **App Name**: Evergreeners

## Core Features:

- GitHub OAuth: Securely authenticate users via GitHub OAuth for seamless data integration.
- Streak Calculation: Automatically calculate and update user streaks based on daily GitHub activity using mock data.
- Activity Dashboard: Visualize user activity with streak counters, sparklines, heatmaps and charts, all driven by mock data.
- Insight Engine: A tool using generative AI to extract textual insights from your activity data, providing you with prompts and feedback.
- Goal Setting: Allow users to define custom coding goals.
- Profile Management: Enable users to customize their profiles and manage privacy settings.
- Dark/Light Mode: Toggle button for switching between light and dark modes, with system setting auto-detection.

## Style Guidelines:

- Primary color: Forest Green (#228B22) for primary actions, conveying growth and activity.
- Background color: Sage (#D8E2DC) for subtle backgrounds, offering a calming and unobtrusive feel. Forest Green (#228B22) or Slate-950 can be used for dark mode backgrounds.
- Accent color: Lime (#A7D14B) for high-energy accents, signaling new records or milestones.
- Body and headline font: 'Inter' (sans-serif) for clean and modern UI.
- Code font: 'Source Code Pro' (monospace) for clear display of coding related numbers and streaks.
- Use consistent, nature-inspired icons. Use Lucide React icons.
- Grid-based layout with clear sections for streaks, activity, and goals. Implement a persistent Left Sidebar for navigation.
- Subtle animations to indicate activity and progress.
